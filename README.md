# About

**EscNet** is an open-source micro framework built on **.NET Standard**,
making it compatible with both:

-   .NET Core / .NET
-   .NET Framework

<br>
The goal of EscNet is to **simplify everyday development tasks**
commonly required in enterprise applications, such as:

-   Email sending
-   Cryptography utilities
-   Hashing algorithms
-   Logging
-   Security utilities

The framework is continuously evolving and new features are under
development.

# Installation

EscNet is available on **NuGet**.

### .NET CLI

``` bash
dotnet add package EscNet
```

### Package Manager

``` powershell
Install-Package EscNet
```

Or access directly via NuGet:

https://www.nuget.org/packages/EscNet/

# Features

Currently the framework includes the following modules:

### Email

-   SMTP email sending utilities

### Cryptography

-   Rijndael encryption
-   Caesar cipher

### Hashing Algorithms

-   SHA1
-   Argon2Id
-   SCrypt
-   BCrypt

<br>
Additional features such as **logging and reporting utilities** are
under development.
